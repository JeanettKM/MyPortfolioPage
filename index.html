Social media page, 2nd year Javascript project

This is a social media page made for the second year Javascript project at school. It was made with HTML5, CSS3 and Javascript. It is responsive and takes use of the API provided by Noroff

  • The project shows posts fetched from the API provided by Noroff.

  • The project lets the user register a new account if provided the correct credentials. If the credentials are wrong, the user will be notified by error messages.

  • The user can log in and out, and the information will be stored and deleted in local storage.

  • The design I chose was fairly simple as I spent most of the time focusing on the code, but simple isn't always a negative thing. For the design, I chose to use Bulma CSS Framework.
